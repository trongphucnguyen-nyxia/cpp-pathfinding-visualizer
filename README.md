# C++ Pathfinding Visualizer

## Overview

C++ Pathfinding Visualizer is an interactive OpenGL application that demonstrates pathfinding algorithms on editable grid-based environments. Users can modify obstacles, set start and goal positions, and visualize different search strategies.

## Features

* Interactive grid environment
* Editable obstacles
* Custom start and goal positions
* Random start and goal generation
* Breadth-First Search
* A* Search
* Best-First Search
* Real-time pathfinding visualization
* OpenGL/GLUT graphical interface

## Technologies Used

* C++
* OpenGL
* GLUT
* Object-Oriented Programming
* Graph Search Algorithms

## Skills Demonstrated

* Algorithm visualization
* Pathfinding and graph search
* C++ development
* GUI/graphics programming
* Event-driven programming
* Data structures
* Software design

## Purpose

This project was created to explore how pathfinding algorithms work in visual environments. It demonstrates the differences between uninformed and informed search strategies while allowing users to interactively modify the search space.
